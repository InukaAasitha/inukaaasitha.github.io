Security Vulnerability Assessment

As a part of Computer Security Module me and my team has done Security Vulnerability Assessment for Rohan Construction Firm situated in Knadana, Sri Lanka. We have found several vulnerabilities in their existing system which are possibility of Injection Attacks for Database, insufficient Logging and Monitoring, Exposed Configuration Files, Weak Login Constrols, Social Engineering and Tailgating. For identified vulnerabilities we have instroduced both short-term and long-term remidiations that could protect their system from intruders and attacker.
